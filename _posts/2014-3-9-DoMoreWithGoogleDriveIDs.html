Ever notice that the URL to a shared file in Google Drive really long? Often, when I need to share that url with a large group of people at a training or conference, I need to shorten it. I prefer Bitly.com because I can customize the urls and make QR codes, and get analytics (More on that at a later post).

There is some very useful information in this url. Prominently is the file's identification key. This is a series of about 44 letters, numbers, and sometimes symbols that help Google keep all the files saved in Drive by everyone separate and to locate your file when you call it. Think of it like a combination lock where you have the spin the dial 44 times and there are 58+ characters on the dial (letters are case sensitive, so lower and upper case matters).

Example URL of am image file saved in Google Drive

https://drive.google.com/#folders/0Bxriiaw-6rI1TU1rN3FmR1EtZHc 

fileID=0Bxriiaw-6rI1TU1rN3FmR1EtZHc

Example URL of a document file saved in Google Drive

https://docs.google.com/document/d/1PYB_miC9xfaC-16xY-47kAKnTkzGHGV1mCM_xNxBpTI

docID=1PYB_miC9xfaC-16xY-47kAKnTkzGHGV1mCM_xNxBpTI

Example URL of a spreadsheet file saved in Google Drive

Notice how after the ID the #gid=0 tells you it will open the first sheet in the spreadsheet. #gid=1 would open the 2nd and so on.

https://docs.google.com/spreadsheet/ccc?key=0Ahriiaw-6rI1dHU3VW1nX2EtcjB6MFpEZlo4Q0pNZ0E#gid=0

spreadsheetID=0Ahriiaw-6rI1dHU3VW1nX2EtcjB6MFpEZlo4Q0pNZ0E


Some things you can do with this key:

Import data from other spreadsheets.

=importrange("driveKey","sheetName!A1:Z10")
NOTE: Quotation marks are required, Exclamation Point required after the name of the sheet.

Create a link to download the PDF version of the file on our website.

https://docs.google.com/document/d/documentId/export?format=pdf

Google Apps Script to retrieve values from a file or append data to a file. 

 var getSheet = SpreadsheetApp.openById("sheetId")

var getDocument = DocumentApp.openById("documentId")

Insert Images or Video into a website from drive.

img src="http://googledrive.com/host/folderId/imageID"
video src="http://googledrive.com/host/folderId/imageId"
NOTE: The file must be Shared With "Anyone With a Link" in order for this to work.
